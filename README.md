## Permisos en archivos y carpetas
### Administrar usuarios y grupos

```bash
groups usuario #saber si un usuario pertenece a un grupo
```


### Permiso a carpeta y direcctorios

```bash
chmod 700 carpeta #asigna permisos con codigo a una carpeta.
chmod rwx------ carpeta # 
```
